# 🎯 TrackMaster – Gamified Todo App

A fully client-side productivity application built with **HTML, CSS, and JavaScript**.  
Manage tasks with priorities and deadlines, track your progress with a live progress bar, earn XP and badges, and switch between light/dark themes — all data stored locally in your browser.

## 🚀 Features

- 🔐 **User Authentication**  
  Sign up / login system (demo account included)

- ✅ **Task Management**  
  Create, edit, and delete tasks with:
  - Title
  - Description
  - Category
  - Priority
  - Deadline

- 📊 **Progress Tracking**  
  Visual progress bar and statistics (completed vs pending)

- 🔔 **Smart Reminders**  
  Highlights tasks due within the next 7 days

- 🎮 **Gamification System**  
  - Earn XP for completed tasks  
  - Unlock ranks (Beginner → Legend)  
  - Collect achievement badges  

- 🌓 **Light / Dark Mode**  
  Toggle theme with saved preferences via `localStorage`

- 📁 **Categories & Organization**  
  Group tasks into Work, Personal, Study, Health, etc.

- 📋 **Priority Sorting**  
  Tasks automatically sorted by priority and deadline

- 💾 **Offline First**  
  No backend required — everything runs in your browser
