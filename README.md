# 🎯 TrackMaster – Gamified Todo App

A fully client‑side productivity application built with **HTML, CSS, and JavaScript**.  
Manage tasks with priorities and deadlines, track your progress with a live progress bar, earn XP and badges, and switch between light/dark themes – all data stored locally in your browser.

## 🚀 Features

- 🔐 **User Authentication** – Sign up / login (demo account included)
- ✅ **Task Management** – Create, edit, delete tasks with title, description, category, priority, and deadline
- 📊 **Progress Bar & Statistics** – Visual overview of completed vs pending tasks
- 🔔 **Deadline Reminders** – Highlights upcoming tasks due within the next week
- 🎮 **Gamification** – Earn XP for completing tasks, unlock ranks (Beginner → Legend) and earn badges
- 🌓 **Light / Dark Mode** – Theme toggle with persistent preference saved in `localStorage`
- 📁 **Project Categories** – Organise tasks by Work, Personal, Study, Health, etc.
- 📋 **Priority Ordering** – Tasks automatically sorted by priority (High → Medium → Low) and deadline
- 💾 **Offline‑First** – All data stays in your browser; no server required
